Development environment setup

Setup PostgreSQL

  1. Install Postgre SQL
    You can download and install the latest version of Postgres SQL from http://www.enterprisedb.com/products/pgdownload.do#windows.
    To run the below command at command prompt, you should set the path of PostgreSQL\bin to the PATH environment variable of the Windows.
    You need php-pdo-pgsql to be installed into PHP, too.
  2. Create the database simple_equipment_management for the current system by running the following command at the command prompt.
    createdb -U postgres -E UTF8 simple_equipment_management
    On linux, you need run the command as user postgres sudo -u postgres createdb -U postgres -E UTF8 simple_equipment_management

    To delete the database

    dropdb -U postgres simple_equipment_management
    On linux, you need run the command as user postgres sudo -u postgres dropdb -U postgres simple_equipment_management
  3. Create the user simple_equipment_management
    1. Run PostgreSQL client at the command prompt
      psql -U postgres
    2. Run the following PostgreSQL commands to create the user and grant privileges
      create user simple_equipment_management with password 'simple_equipment_management';
GRANT ALL PRIVILEGES ON DATABASE simple_equipment_management TO simple_equipment_management;
  4. To import the data schema (don't do this. Use migration instead)
    
psql -U simple_equipment_management -d simple_equipment_management -f schema.pgsql.sql
